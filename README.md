# linux-perf-tools

This repository contains a collection of tools that I am creating to process various performance data collected from a GNU/Linux server, primarily from Sysstat SAR.

Consider this a work in process

---

## Contents
| Directory/File                     | Description                                                    |
|------------------------------------|----------------------------------------------------------------|
| `perf_capture_requirements`        | Contains documentation on how to collect perf data to process  |


## Getting Started

Follow the information in perf_capture_requirements to start pulling the required performance data that these tools will use to generate information from.
